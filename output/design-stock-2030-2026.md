# Design Brief: 20~30대 주식 인식 및 참가실태 2026ver.

결론:
1. 톤앤매너는 “장식 없는 다크 리포트 캐러셀”이다.
2. 사진·일러스트·오브·블롭 없이 숫자, 선, 막대, 표, 체크리스트만 사용한다.
3. 모든 카드 오른쪽 상단에는 실제 EARLYSHINE 투명 로고를 넣고, 마지막 7장은 로고 엔딩 카드로 만든다.

## 톤앤매너

투자 리포트를 인스타 캐러셀로 압축한 느낌. 차갑지 않게 보이되, 감성 장식 없이 정보 위계를 또렷하게 둔다.

## 컬러

- Base: `#0F0E0C`
- Panel: `#16140F`
- Ink: `#F3EDDE`
- Muted: `#9F927A`
- Tan: `#D4B896`
- Orange: `#D96035` 아주 제한적으로 경고/핵심 숫자에만 사용
- Line: `#514533`

## 폰트

- Pretendard OTF 사용.
- 제목: Pretendard Black 66~78px.
- 숫자: Pretendard Black 54~86px.
- 본문: Pretendard Regular/Bold 20~28px.
- 출처/주의: 13~15px.
- PyMuPDF 렌더에서 `%` 글리프가 깨질 수 있으므로 큰 수치에는 `%` 기호를 붙이지 않고, 차트/패널 제목에 `단위: 퍼센트` 또는 `응답 비율`을 명시한다.

## 카드별 레이아웃

1. Hook: 상단 큰 제목 + 하단 3개 숫자 카드.
2. Participation: 왼쪽 제목, 오른쪽 연령대별 가로 막대.
3. Perception: 기대/경계 2열 비교 패널.
4. Portfolio: 세대별 국내/해외 비중 막대 + 해석 문장.
5. Channel Risk: 4단계 정보 흐름 + 국내 신호/해외 경고 비교 + 검수 질문 바.
6. Checklist: 6개 체크 항목의 2열 그리드.
7. Ending: 중앙 EARLYSHINE 로고.

## 카드 3 재발 방지 메모

카드 3처럼 설문 인식 수치를 비교하는 장표는 `큰 숫자 카드`를 단순 반복하면 안 된다. 표처럼 보이지만 열 헤더, 행 의미, 단위, 비교 기준이 약해져 사용자가 무엇을 읽어야 하는지 놓친다.

다음 제작부터 `Perception` 카드에는 아래 4요소를 반드시 둔다.

- 컬럼 제목: `기대`, `경계`처럼 비교 축을 명확히 표시한다.
- 문항 라벨: 각 숫자가 어떤 설문 문항의 응답인지 숫자 바로 옆에 둔다.
- 단위/기준: `%` 글리프 문제가 있으면 숫자에 붙이지 말고 컬럼 또는 표 헤더에 `응답 비율, 단위: 퍼센트`를 둔다.
- 핵심 해석: 표 하단에 `기대는 높지만 시장 신뢰는 낮다`처럼 읽어야 할 결론을 한 줄로 넣는다.

금지:
- `compare_metric()` 같은 숫자 카드 컴포넌트를 6개 쌓아 표처럼 쓰지 않는다.
- 외곽 패널과 내부 행 박스를 모두 진하게 그려 이중 테두리 표처럼 만들지 않는다.
- 단위를 카드 상단 설명에만 두고 각 표 영역에서 사라지게 하지 않는다.
- 모바일 축소 시 숫자만 남는 장표로 만들지 않는다.

## 금지 스타일

- 원형 오브, 그라데이션 블롭, 배경 장식용 점무늬.
- 귀여운 캐릭터, 손그림, 파스텔 노트 템플릿.
- 근거 없는 상승 화살표 남발.
- 텍스트 `EARLYSHINE` 대체 로고.

## QA 체크리스트

- 모든 숫자에 출처 또는 기준 연도가 붙어 있는가?
- 2030을 시장 최대 큰손처럼 과장하지 않았는가?
- PNG/PDF에서 한글이 깨지지 않는가?
- 카드 7장이 같은 시스템으로 보이는가?
- 텍스트가 1080x1350 안전 영역 밖으로 나가지 않는가?
- 패키징 이후 `design-review`가 실제 PNG 기준으로 P0/P1/P2 검수를 수행했는가?
- 수치만 남은 카드 없이 해석 문장이 같이 보이는가?
- 표형 장표에 열 헤더, 문항 라벨, 단위/기준, 핵심 해석이 모두 보이는가?
