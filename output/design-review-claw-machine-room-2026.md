# Design Review: claw-machine-room-2026

역할: design-review / Beauvoir  
검수일: 2026-06-18  
검수 대상: `output/cards/claw-machine-room-2026-card-01.png` ~ `card-07.png`, `output/claw-machine-room-2026-render-check.png`, `output/claw-machine-room-2026-cardnews.pdf`, `output/claw-machine-room-2026-cardnews.html`

결론:
1. 추가 시각 QA에서 카드 04의 중앙 브리지와 하단 아이콘이 패널 경계를 침범하던 문제가 발견되어 수정했습니다.
2. 수정 후 카드 04는 `취향 수집` 라벨, 좌우 패널, 캡슐/박스 아이콘이 서로 겹치지 않고 패널 안에 정리됩니다.
3. 현재 통과 여부는 통과입니다. 7장 구성, 1080x1350 PNG, PDF, 실제 EARLYSHINE 로고 이미지 삽입은 정상 확인했습니다.

근거:
- 산출물 확인: 카드 PNG 7장 모두 `1080x1350`이며, `render-check.png`에서 7장 전체가 최신 세트로 보입니다.
- 재검수 확인: `card-04.png`, `render-check.png`, PDF, HTML이 2026-06-18 오전 2:28 기준으로 갱신되어 있습니다.
- PDF 확인: `claw-machine-room-2026-cardnews.pdf`는 6,978,119 bytes, PDF 내부 `/Type /Page` 기준 7페이지로 확인했습니다.
- HTML 확인: 카드 크기는 `1080px x 1350px`, `@page`도 동일 크기입니다. 로고는 텍스트 대체가 아니라 `../assets/logo/main_a_transparent.png` 실제 이미지로 삽입되어 있습니다.
- 로고 파일 확인: `assets/logo/main_a_transparent.png`는 `2032x250`, `Format32bppArgb` 투명 PNG입니다.
- 전체 디자인: 검정 배경, 크림 타이포, 금색/오렌지 라인, 실제 로고 사용이 일관적입니다. 장식선은 강하지 않아 메시지를 크게 방해하지 않습니다.
- 메시지 흐름: 01 Hook, 02 경험 구조, 03 국내 확산, 04 해외 수집 시장, 05 가드레일, 06 체크리스트, 07 엔딩으로 흐름은 자연스럽습니다.
- 카드 04 재검수: 좌측 박스의 `일본 캡슐토이 시장`, 우측 박스의 `Pop Mart 매출` 라벨이 수치의 출처/성격을 분명히 하고, 하단 `관심도 신호` 박스가 언급량 지표를 별도 맥락으로 분리합니다.
- 카드 04 추가 시각 QA: 기존 렌더에서는 중앙 브리지 박스가 좌우 패널 상단 경계와 어색하게 겹치고, 하단 캡슐/박스 아이콘 일부가 패널 밖으로 침범했습니다. 현재 렌더에서는 브리지 라벨을 패널과 분리하고, 아이콘 개수·좌표·크기를 조정해 모든 요소가 패널 내부에 머뭅니다.

## P0

없음.

## P1

해결 완료.

1. 카드 04: 해외 수치의 정체가 카드 안에서 즉시 잡히지 않는 문제는 해결됐습니다.
   - 확인: 좌측 박스에 `일본 캡슐토이 시장`, 우측 박스에 `Pop Mart 매출` 라벨이 추가됐고, 하단 언급량은 `관심도 신호`로 분리됐습니다.
   - 판단: 숫자가 같은 기준의 단순 비교처럼 보이는 위험이 줄었고, "랜덤이 수집 시장이 됐다"는 제목과 근거가 자연스럽게 연결됩니다.

## P2

1. 카드 03: 숫자 위계가 약간 분산됩니다.
   - 현상: `전국 인형뽑기방 6,144곳`과 `청소년게임제공업소 5,957곳`이 모두 크게 보여, 어떤 숫자가 핵심 근거인지 순간적으로 헷갈릴 수 있습니다.
   - 수정 방향: `6,144곳`을 주지표로 유지하고, `5,957곳`은 보조 근거 박스 또는 푸터성 설명으로 낮추면 좋습니다.

2. 카드 03~06: 푸터/출처 문구의 모바일 가독성이 낮습니다.
   - 현상: 1080px 원본에서는 읽히지만, 인스타 피드/모바일 뷰에서는 출처와 주의 문구가 지나치게 작아질 가능성이 큽니다.
   - 수정 방향: 푸터 글자 크기를 소폭 키우고 대비를 조금 올리되, 본문 시선보다 강해지지 않게 조정해 주세요.

3. 카드 06: 체크리스트 영역의 하단 밀도가 높습니다.
   - 현상: 핵심 결론은 좋지만, 하단 체크 4개와 CTA가 좁은 영역에 몰려 있습니다.
   - 수정 방향: CTA 버튼을 조금 줄이거나 체크 항목 간 세로 간격을 정리하면 저장용 체크리스트로 더 편하게 읽힙니다.

4. 카드 02: 좌측 하단 `PRICE RESULT COLLECT` 보조 그래픽의 정보성이 약합니다.
   - 현상: 오른쪽 3단 구조는 명확한데, 좌측 하단 박스는 읽어도 추가 이해가 거의 늘지 않습니다.
   - 수정 방향: 유지해도 큰 문제는 없지만, 카드 02의 핵심인 `가격 -> 결과 확인 -> 소장품` 순환을 더 직접적으로 보여주는 화살표/루프 요소가 더 효과적입니다.

## 카드별 판단

- 카드 01: 통과. 훅이 강하고, "작은 시도 -> 바로 확인 -> 손에 잡히는 보상" 구조가 한눈에 보입니다.
- 카드 02: 통과. 운이 아니라 경험 구조라는 메시지가 명확합니다. 보조 그래픽만 P2 개선 여지가 있습니다.
- 카드 03: 조건부 통과. 국내 확산 메시지는 선명하지만 보조 숫자 위계 조정이 있으면 더 좋습니다.
- 카드 04: 통과. 수치 라벨과 지표 성격이 보강되어 메시지 전달력이 안정됐습니다.
- 카드 05: 통과. 재미와 운영 리스크를 균형 있게 보여주며, 경품 기준/민원 리스크 구성이 명확합니다.
- 카드 06: 통과. 최종 메시지가 가장 실용적으로 닫힙니다. 하단 밀도만 소폭 조정 권장입니다.
- 카드 07: 통과. 실제 로고 중심 엔딩 카드로 정상입니다.

다음 단계:
- 최종 배포 가능합니다. 카드 03/06 및 푸터 가독성 관련 P2는 여유가 있을 때 다듬으면 더 좋아지는 선택 개선 사항입니다.
