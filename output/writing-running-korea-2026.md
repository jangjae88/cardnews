## 결론

1. 최종 프레임은 **러닝은 운동을 넘어 러너가 반복해서 지나는 생활권 동선 비즈니스**로 잡는다.
2. 카드별 문구는 큰 제목 1개, 보조 문장 1개, 칩/도표 문구 중심으로 짧게 구성한다.
3. 확인 필요 숫자는 쓰지 않고, 확인된 수치도 카드 2의 작은 근거 칩에만 제한적으로 둔다.

## 카드별 최종 문구

### 01 / 07 RUNNING KOREA 2026

- 제목: **러닝은 무료지만, 길목엔 시장이 붙는다**
- 보조 문장: 크루, 대회, 앱, 매장, 회복제품이 러너의 하루 동선 안으로 들어옵니다.
- 동선 칩: `크루` / `대회` / `앱` / `매장` / `회복제품`
- 보조 라벨: `모인다` / `접수한다` / `기록한다` / `들른다` / `회복한다`
- 푸터 주의 문구: 프레임 카드입니다. 추정 기반 인구·시장 규모 수치는 제외했습니다.

### 02 / 07 CULTURE

- 제목: **혼자 뛰던 러닝이 모이는 경험이 됐다**
- 보조 문장: 러닝크루는 생활체육 언어 안으로 들어오고, 대회는 참가 경험 자체가 주목받습니다.
- 4분면 문구: `생활체육` / `러닝크루` / `대회 접수` / `참가 경험`
- 근거 칩: `생활체육 참여율 62.9%` / `전년 대비 +2.2%p` / `JTBC 서울마라톤 30,000명` / `대구마라톤 약 41,254명`
- 푸터 주의 문구: 출처: 2025 국민생활체육조사, 정책브리핑, JTBC 서울마라톤, 대구마라톤 보도.

### 03 / 07 PLATFORM

- 제목: **기록은 러너의 소속감 카드가 된다**
- 보조 문장: 운동 로그가 접수, 인증, 커뮤니티를 잇는 작은 신분증처럼 쓰입니다.
- 흐름도 문구: `러닝 기록` -> `디지털 카드` -> `접수/인증` -> `커뮤니티`
- 근거 배지: `카카오-러너블 러너스 카드` / `Runable 접수`
- 푸터 주의 문구: 플랫폼 효과를 참여 증가나 매출 증가로 단정하지 않습니다.

### 04 / 07 PLACE

- 제목: **러너가 지나는 곳에 거점이 생긴다**
- 보조 문장: 피팅, 측정, 보관, 물/간식, 휴식, 크루 모임이 코스 주변에 붙고 있습니다.
- 미니맵 핀: `피팅` / `측정` / `보관` / `물·간식` / `휴식` / `크루 모임`
- 근거 패널: `더현대 러닝 클럽` / `러닝 스페셜티 스토어` / `CU 한강 러닝 스테이션`
- 푸터 주의 문구: 사례 기반 흐름입니다. 모든 매장으로 일반화하지 않습니다.

### 05 / 07 PRODUCT

- 제목: **러닝제품은 전후 루틴으로 넓어진다**
- 보조 문장: 신발 하나가 아니라 준비, 기록, 보급, 회복의 카테고리로 쪼개집니다.
- 매트릭스 열 제목: `Before` / `During` / `After`
- 시간 라벨: `뛰기 전` / `뛰는 중` / `뛴 뒤`
- Before 항목: `러닝화 피팅` / `의류` / `양말` / `조끼`
- During 항목: `웨어러블` / `기록 앱` / `수분 보충` / `에너지 간식`
- After 항목: `리커버리 슈즈` / `회복용품` / `발 건강` / `보급·영양`
- 푸터 주의 문구: 특정 제품의 효능, 부상 예방, 매출 전망은 단정하지 않습니다.

### 06 / 07 CHECKLIST

- 제목: **숫자보다 먼저, 동선을 보세요**
- 보조 문장: 내 고객이 어디서 모이고, 뛰고, 쉬고, 반복 구매하는지 확인해야 합니다.
- 좌측 패널 제목: `기회`
- 기회 체크: `크루가 모이는 장소인가` / `뛰기 전후 들를 이유가 있는가` / `기록·인증과 연결되는가` / `작은 반복 구매가 가능한가`
- 우측 패널 제목: `주의`
- 주의 체크: `장비 과열을 부추기지 않는가` / `부상·안전을 과장하지 않는가` / `확인 안 된 숫자를 쓰지 않는가`
- 하단 캡슐: `SAVE THIS` / `RUNNER ROUTE CHECK`
- 푸터 주의 문구: 성과 보장이나 창업 추천이 아닌, 러너 동선 검토 기준입니다.

### 07 / 07 EARLYSHINE

- 본문 문구: 없음
- 중앙 요소: 실제 EARLYSHINE 로고 이미지
- 푸터 문구: `07 / 07`
- 패키징 주의: 텍스트 로고로 대체하지 말고 `assets/logo/main_a_transparent.png`를 사용합니다.

## 제외한 위험 표현

- 공식 통계로 확정되지 않은 러닝 인구 표현.
- 원 리포트 확인이 필요한 시장 규모 표현.
- 원사이트 검증 전 대회 수를 확정처럼 보이게 하는 표현.
- 검색량, 거래액, 브랜드 매출 증가를 원자료 없이 강조하는 표현.
- 성과, 판매, 건강 결과를 보장하는 표현.

- 카드 3은 실제 앱 화면이나 실제 브랜드 로고를 복제하지 않고 추상 UI로 구현한다.
