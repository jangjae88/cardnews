# Writing: 20~30대 주식 인식 및 참가실태 2026ver.

결론:
1. 문구는 짧고 단정하게 썼다.
2. “수익 보장”, “무조건”, “답이다” 같은 과장 표현은 제외했다.
3. 각 카드 푸터에는 출처와 기준을 압축해 넣는다.

## 최종 카드 문구

### 01

- 제목: 2030은 주식을 안 하는 세대가 아니다
- 보조: 더 빨리 시작하고, 더 넓게 보고, 더 불안하게 판단하는 세대다.
- 칩: 1,456만 주주 / 2030 약 28% / 20대 해외 ETP 60%

### 02

- 제목: 국장은 아직 4050이 크다
- 보조: 그래도 30대는 개인 주주 수 기준 세 번째 축이다.
- 표 문구: 50대 23.1 / 40대 21.8 / 30대 19.1 / 60대 15.3 / 20대 8.8

### 03

- 제목: 돈은 벌고 싶지만 시장을 다 믿진 않는다
- 보조: 2026년 투자 심리는 기대와 경계가 같은 화면에 뜬다. 단위는 퍼센트다.
- 기대: 20대 49.0, 30대 54.5, 주식투자 의향 76.4
- 경계: 돈 있는 사람의 경제활동 62.8, 투자 과열 50.2, 누구나 돈 벌 기회 35.6

### 04

- 제목: 2030의 화면은 국장 밖까지 열려 있다
- 보조: 해외 ETF/ETP는 젊은 투자자의 기본 선택지가 됐다.
- 데이터: 20대 해외 ETP 60.0%, 국내주식 30.8% / 30대 해외 ETP 45.5%

### 05

- 제목: 공부는 빨라졌지만 검증은 느릴 수 있다
- 보조: 유튜브, SNS, 커뮤니티는 입문 장벽을 낮추지만 FOMO와 과신도 같이 키운다.
- 흐름: 수익 인증 → 영상/요약 → 커뮤니티 → 매수 버튼
- 국내 신호: 22.0 / 20대는 SNS 수익 인증에 더 쉽게 자극받는다.
- 해외 경고: 61.0 / 35세 미만 투자자는 핀플루언서 추천을 실제 결정에 쓴다.
- 결론 바: 검수 질문: 출처가 보이나? 손실도 말하나? 상품 구조를 설명하나?

### 06

- 제목: 2030 주식 콘텐츠는 검증 구조가 먼저다
- 보조: 숫자보다 먼저 봐야 할 6가지.
- 체크: 출처 날짜 / 표본 대상 / 국내·해외 구분 / 수익률·손실 동시 표기 / 상품 구조 / 확정 표현 금지
