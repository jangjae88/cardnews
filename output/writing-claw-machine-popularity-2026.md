## 결론

1. 최종 프레임은 **운이 아니라, 낮은 진입가와 즉시 결과, 소장 보상이 만든 짧은 보상 루프**로 확정했습니다.
2. 본문 6장은 Hook -> Problem -> Domestic -> Global -> Guardrail -> Takeaway 흐름으로 구성했습니다.
3. 작은 브랜드 관점의 결론은 **랜덤을 따라 하기보다 보상 경험을 작고 투명하게 설계하라**입니다.

## 카드별 최종 문구

### 01 / 07 HOOK

- 제목: **뽑기방은 왜 다시 눈에 띌까?**
- 보조 문장: 큰 소비보다 작은 돈으로, 바로 확인하고 손에 쥘 수 있는 재미가 있습니다.
- 하단 루프 문구: `작은 시도` -> `바로 확인` -> `손에 잡히는 보상`
- 보조 칩: `소액` / `즉시성` / `소장감`
- 푸터 주의 문구: 모든 지역·연령의 유행으로 일반화하지 않고, 관찰된 소비 흐름으로 봅니다.

### 02 / 07 EXPERIENCE

- 제목: **핵심은 운 하나가 아닙니다**
- 보조 문장: 한 번 더 해볼 만한 가격, 바로 보이는 결과, 남는 물건이 함께 작동합니다.
- 3분할 패널:
  - `가격` / 부담이 낮아야 첫 시도가 생깁니다
  - `결과 확인` / 기다리지 않아야 재미가 끊기지 않습니다
  - `소장품` / 끝나고도 남는 물건이 있어야 기억됩니다
- 콜아웃: `운보다 경험 설계`
- 푸터 주의 문구: 심리 효과를 과도하게 해석하지 않습니다. 보상 구조를 설명하는 카드입니다.

### 03 / 07 DOMESTIC

- 제목: **국내에선 문턱이 낮아졌습니다**
- 보조 문장: 매장 수 증가, 무인 운영, 카드 결제가 가벼운 놀이 소비를 더 쉽게 만들었습니다.
- 대형 KPI: `6,144곳`
- KPI 보조 문구: `2025년 3분기 전국 인형뽑기방`
- 비교 칩: `전년 동기 5,028곳` / `+1,116곳`
- 문턱 스택: `무인 운영` / `카드 결제` / `가성비 놀이문화`
- 보조 근거 칩: `청소년게임제공업소 5,957곳` / `최근 2년 20% 가까이 증가`
- 푸터 주의 문구: 출처: 경기일보/Daum, 연합뉴스. 증가 수치는 업소 수 기준이며 수요 증가를 뜻하지 않습니다.

### 04 / 07 GLOBAL

- 제목: **해외에선 랜덤이 수집 시장이 됐습니다**
- 보조 문장: 일본 캡슐토이와 Pop Mart 흐름은 사람들이 취향을 모으는 방식을 보여줍니다.
- 좌측 패널: `일본 캡슐토이` / `2025년 약 1,960억엔` / `2024년 대비 +39.0%`
- 우측 패널: `Pop Mart` / `2025년 매출 371.2억 위안` / `+184.7%`
- 중앙 브리지 라벨: `취향 수집`
- 보조 관심 칩: `2025년 가챠 언급량 810,583건` / `전년 대비 약 +82%`
- 푸터 주의 문구: 출처: JACTA, Pop Mart, 제일기획 매거진. 해외 사례를 국내 시장과 직접 동일시하지 않습니다.

### 05 / 07 GUARDRAIL

- 제목: **재미가 커질수록 선이 필요합니다**
- 보조 문장: 공정성, 경품 기준, 민원 대응이 빠지면 작은 재미가 운영 리스크가 됩니다.
- 리스크 게이트:
  - `공정성` / 확률·난이도 오해를 줄일 것
  - `경품 기준` / 허용 기준을 확인할 것
  - `민원 리스크` / 기계·결제·환불 문의를 관리할 것
- 기준 칩: `경품 기준 10,000원` / `2020년 시행령 개정`
- 경고 칩: `경품기준 위반 보도` / `기계 개조 의혹 보도` / `민원 증가 보도`
- 푸터 주의 문구: 출처: 정책브리핑, 연합뉴스. 위반·개조는 전체 업소 단정이 아니라 보도된 리스크입니다.

### 06 / 07 TAKEAWAY

- 제목: **따라 할 건 랜덤이 아니라 보상 설계**
- 보조 문장: 작은 브랜드라면 뽑기보다, 고객이 다시 참여할 이유를 작고 투명하게 만들어야 합니다.
- Before 패널 제목: `그대로 따라 하면 위험`
- Before 항목: `랜덤만 강조` / `규칙 불명확` / `보상 매력 부족`
- After 패널 제목: `가져올 것은 설계`
- After 항목: `낮은 진입가` / `모으고 싶은 보상` / `바로 공유할 장면`
- 체크리스트:
  1. `진입 가격이 가벼운가`
  2. `보상이 소장할 만한가`
  3. `규칙과 조건이 투명한가`
  4. `재방문할 이유가 있는가`
- 하단 캡슐: `SAVE THIS` / `REWARD DESIGN CHECK`
- 푸터 주의 문구: 랜덤 방식은 성과를 약속하지 않습니다. 업종별 법규, 고지, 환불·교환 기준 확인이 필요합니다.

### 07 / 07 EARLYSHINE

- 본문 문구: 없음
- 중앙 요소: 실제 EARLYSHINE 로고 이미지
- 푸터 문구: `07 / 07`
- 패키징 주의: 텍스트 로고로 대체하지 말고 `assets/logo/main_a_transparent.png`를 사용합니다.

## 사용하지 않은 표현 유형

- 특정 심리 상태를 단정하는 표현.
- 사행성 업종과 직접 동일시하는 표현.
- 기계 운영 상태를 근거 없이 단정하는 표현.
- 특정 세대 전체를 하나의 취향으로 묶는 표현.
- 랜덤 이벤트의 성과를 약속하는 표현.
- 고가 경품과 참여율을 단순 연결하는 표현.
- 해외 랜덤 굿즈 시장 성장을 국내 뽑기방 성장으로 바로 연결하는 표현.

## Packaging 사용 주의

- 카드 1은 질문형 훅이므로 숫자를 크게 넣지 않습니다.
- 카드 2는 `가격 / 결과 확인 / 소장품` 3개 패널이 한눈에 보이게 합니다.
- 카드 3의 `6,144곳`은 기준·출처·해석 문장과 붙여 배치합니다.
- 카드 4는 실제 캐릭터, Pop Mart 로고, 제품 외형을 재현하지 않고 텍스트와 추상 라인워크만 씁니다.
- 카드 5는 겁주는 표현보다 운영 전 체크해야 할 게이트처럼 읽히게 합니다.
- 카드 6은 창업 권유나 랜덤 판매 추천이 아니라 보상 설계 체크리스트로 닫습니다.
