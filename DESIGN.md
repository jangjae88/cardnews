# DESIGN.md

## 기준

이 프로젝트의 카드뉴스 디자인은 `EARLYSHINE_에디토리얼_디자인가이드_v1.pdf`의 Warm / Tan 라인을 기본 시각 언어로 사용한다.

참고 복사본:
- `output/earlyshine-editorial-design-guide-v1.pdf`
- `output/earlyshine-guide-pages/guide-page-01.png`
- `output/earlyshine-guide-pages/guide-page-02.png`

## 디자인 담당

- 총괄 판단: `team-lead`
- 디자인 시스템/비주얼 콘셉트: `design`
- 이미지 소재 생성: `image`
- HTML/CSS 구현 및 PNG/PDF 출력: `packaging`

`design`은 디자인 시스템을 정하고, `image`는 사진 소재만 만들며, `packaging`은 디자인을 구현한다. `packaging`은 새로운 톤을 임의로 만들지 않는다.

## 디자인 한 줄 정의

웜 니어블랙 배경 위에 탄 라인워크와 아이보리 타이포그래피로 구성한, 컴팩트한 비즈니스 매거진형 인스타 카드뉴스.

## 컬러

- Base: `#0F0E0C` 페이지 배경, 웜 니어블랙
- Panel: `#16140F` 카드·콜아웃 배경
- Ink: `#F3EDDE` 제목·본문, 웜 아이보리
- Muted: `#9F927A` 보조 텍스트·캡션
- Tan: `#D4B896` 핵심 강조·브랜드 시그니처
- Tan Deep: `#B79A6E` 라인워크·서브 강조
- Good: `#CDB47F` 긍정 표식, 절제 사용
- Bad: `#C36A5C` 금지 표식, 절제 사용

금지:
- 오렌지 CTA 중심 디자인
- 보라색 그라데이션
- 파스텔 노트 템플릿
- 귀여운 툰/손그림
- 장식용 블롭/오브

## 타이포그래피

- 기본 서체: Pretendard Variable
- 제목: 900 Black, tracking `-0.01em`
- 섹션 국문: 800 ExtraBold
- 영문 라벨: 600~700, `0.26em~0.30em`, uppercase
- 본문: 400~600, line-height `1.5~1.65`
- 메타/푸터: 800, `0.22em`

권장 크기:
- 메인 제목: 76~104px
- 서브 설명: 24~30px
- 표/칩 텍스트: 18~26px
- 주석/출처: 14~18px

## 레이아웃

- 카드 수: 6장
- 출력 크기: 1080 x 1350
- 비율: 4:5 세로형 인스타 업로드용
- 카드 내부 안전 여백: 58px 기준
- 섹션 간격은 균일하게 유지
- 라벨과 콘텐츠 사이 간격은 라벨 높이만큼 확보
- 푸터 상단에는 탄 계열 분할선을 둔다
- 로고/워드마크는 우상단에 작고 타이트하게 둔다

## 컴포넌트

- Header lockup: 좌측 `번호 + 넓은 자간 영문 라벨`, 우측 `EARLYSHINE`
- Section label: 탄 컬러, uppercase, 넓은 자간
- Panel: `#16140F` 배경, Tan 20~40% 보더, 작은 라운드
- Chip: 원형 또는 pill 형태, 텍스트 수직 중앙 정렬
- Callout: 패널 배경 + 탄 보더 + 핵심 메시지 한 줄
- Diagram: 다크 위 탄 라인워크, 필요시 점선/화살표 사용
- Footer: 탄 분할선 + 좌측 출처/주의 문구 + 우측 페이지 번호

## 카드 구성 원칙

- 한 장에는 하나의 핵심 메시지만 둔다
- 큰 제목 1개 + 시각 장치 1개를 기본 구조로 한다
- 텍스트보다 구조가 먼저 보여야 한다
- 그래프, 비교표, 매트릭스, 흐름도, 체크리스트를 적극 사용한다
- 사진은 2~3장만 사용하고, 도표 카드와 톤을 맞춘다
- 사진은 실사형이어야 하며, 카드 내부에서 어둡고 따뜻하게 보정한다
- 모든 문구는 카드 프레임 밖으로 나가면 안 된다

## 현재 무인점포 카드뉴스 기준

핵심 프레임:
무인점포는 진짜 무인이 아니다. 직원이 사라진 게 아니라 일이 계산대 밖으로 이동했다.

6장 흐름:
1. Hook: 무인점포, 진짜 무인 아닙니다
2. Growth: 계산 자동화 시장은 커지고 있습니다
3. Backlash: 그런데 일부 매장은 다시 줄였습니다
4. Hidden Cost: 인건비 대신 생기는 비용이 있습니다
5. Role Shift: 사람 역할은 사라지지 않고 이동합니다
6. Checklist: 무인점포 보기 전, 이 6가지를 먼저 보세요

## 최종 QA 체크리스트

- 6장이 EARLYSHINE Warm / Tan 라인으로 일관되는가?
- 우상단 로고 락업이 작고 타이트한가?
- 영문 라벨 자간이 충분히 넓은가?
- 마커·불릿이 옆 텍스트와 수직 중앙 정렬되는가?
- 라벨과 콘텐츠 간격이 답답하지 않은가?
- 패널 보더와 라운드가 과하게 둥글지 않은가?
- 그래프/표/체크리스트가 장식이 아니라 이해를 돕는가?
- PNG/PDF에서 글자가 잘리지 않는가?

## 로고 운영 규칙

- 로고 원본은 `assets/logo/main_a.png`에 보관한다.
- 카드뉴스 렌더링용 투명 배경 로고는 `assets/logo/main_a_transparent.png`를 사용한다.
- 모든 카드의 오른쪽 상단에는 텍스트 `EARLYSHINE` 대신 실제 EARLYSHINE 로고 이미지를 넣는다.
- 카드뉴스 마지막에는 로고 엔딩 카드를 1장 추가한다.
- 엔딩 카드는 중앙에 EARLYSHINE 로고를 크게 배치하고, 불필요한 본문이나 장식은 최소화한다.
- 페이지 표기는 본문 카드와 엔딩 카드를 모두 포함한 총 장수 기준으로 표시한다.

## 추가 디자인 운영 규칙

- 배경에는 큰 원형, 둥근 오브, 블롭, 반투명 원 장식을 사용하지 않는다.
- 배경은 단색을 기본으로 하고, 필요할 때만 거의 티 나지 않는 세로 방향 그라데이션을 사용한다.
- 중앙 정렬 요소는 눈대중으로 맞추지 않고 1080px 캔버스 기준 중심선 `x=540px`에 맞춘다.
- 로고 엔딩 카드, 중앙 노드, 하단 캡슐, 도표 축은 실제 박스 기준으로 수평 중앙 정렬한다.
- 둥근 장식으로 빈 공간을 채우지 않는다. 빈 공간은 정보 위계와 여백으로 남긴다.
